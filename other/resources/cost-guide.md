## Обзор

| Позиция | Стоимость | Обязательно? |
|---------|-----------|--------------|
| Claude Pro | $20/мес | Да |
| DO дроплет (s-2vcpu-4gb) | $24/мес | Блоки 7, 12 |
| Anthropic API ключ | ~$1-5 всего | Только Блок 10 |
| Домен | ~$10/год | Необязательно |

DigitalOcean даёт новым аккаунтам **$200 бесплатного кредита** (60 дней) -- достаточно, чтобы покрыть весь курс.

## Обязательные затраты

### Claude Pro ($20/мес)

- **Включено**: Claude Code в терминале, IDE, десктопе и вебе
- **Модели**: Sonnet (по умолчанию) + Haiku
- **Использование**: тарифицированное, общее с Claude chat -- не безлимитное
- **Opus**: доступен на плане Max ($100/мес), не на Pro
- Подробности на [anthropic.com/pricing](https://anthropic.com/pricing)

### Дроплет DigitalOcean ($24/мес)

- **s-2vcpu-4gb** -- фиксированная рекомендация, не берите меньше
- Новые аккаунты получают **$200 бесплатного кредита** (действует 60 дней)
- **Удалите после курса**, чтобы прекратить оплату -- ваш код хранится в git, дроплет одноразовый

## Какой план мне нужен?

| План | Цена | Что вы получаете в Claude Code |
|------|------|---------------------------------|
| **Pro** | $20/мес | Sonnet + Haiku -- достаточно для всего курса |
| **Max** | $100/мес | Доступ к Opus, повышенные лимиты -- для интенсивного использования |
| **Team** | варьируется | Зависит от типа места -- см. примечание ниже |

<div class="callout-important" markdown="1">

**Командные планы**: стандартные места Team **не** включают Claude Code. Уточните у администратора или ознакомьтесь с [документацией по планам Team от Anthropic](https://support.anthropic.com/en/articles/9267289-how-is-my-team-plan-bill-calculated) перед покупкой.

## Как тратить меньше

- `/compact` почаще -- сжимает разговор, экономит токены
- `--model haiku` для простых задач
- `/clear` между несвязанными задачами
- `--max-turns` в скриптах для ограничения итераций
- `/cost` в сессии для проверки расходов

## Варианты бюджета

### Путь за $20

Только Pro. Пропустите блоки с DO (7, 12). Сосредоточьтесь на изучении возможностей Claude Code, хуков, MCP и агентов локально.

### Путь за $25-45

Полный курс. Pro + дроплет DO. С бесплатным кредитом реальная стоимость ближе к **~$20** за первые два месяца.

### Командный / корпоративный путь

План Max или Team, компания покрывает инфраструктуру. Используйте облачный аккаунт вашей организации для дроплета.
