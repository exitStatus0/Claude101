## Стартовая черта

У каждого инструмента есть момент, когда он «щёлкает». Для `git` это был первый раз, когда вы поняли, что можно откатить что угодно. Для Docker -- первый раз, когда `docker run` поднял базу данных за две секунды. Для Claude Code этот момент наступит прямо сейчас.

Этот блок проведёт вас от нуля до первого настоящего разговора с ИИ-агентом, который умеет читать ваш код, выполнять команды и объяснять найденное. Не автодополнение. Не чат-бот с текстовым полем. Полноценный агент, живущий в вашем терминале.

## Что мы разберём

1. **Что такое Claude Code на самом деле** -- и чем он принципиально отличается от IDE-копайлотов
2. **Установка** -- одна команда, меньше минуты
3. **Аутентификация** -- подключение подписки Anthropic Pro
4. **Первый запуск** -- запускаем `claude` и знакомимся с `/help`
5. **Курсовой проект** -- форкаем `ai-coderrank` и даём Claude увидеть его впервые

## Курсовой проект: ai-coderrank

На протяжении всего курса мы будем работать с **ai-coderrank** -- дашбордом на Next.js 14, который сравнивает ИИ-модели для кодинга по производительности и стоимости. Это реальное приложение с реальной инфраструктурой:

- **Фронтенд**: React 18 + TypeScript + Tailwind CSS + Recharts
- **Бэкенд**: API-роуты Next.js
- **Контейнеризация**: Docker с multi-stage сборкой
- **Оркестрация**: манифесты Kubernetes
- **GitOps**: конфигурация ArgoCD
- **CI/CD**: GitHub Actions воркфлоу

Это не учебный проект-игрушка. Это именно та кодовая база, которую вы встретите на реальной работе -- и идеальная площадка для изучения Claude Code.

## Пререквизиты

- Терминал (macOS Terminal, iTerm2 или любой Linux-терминал)
- Node.js 18+ установлен
- Git установлен и настроен
- Аккаунт на GitHub
- Подписка Anthropic Pro ($20/мес)
