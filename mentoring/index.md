## FAQ

<details>
<summary>Who is this NOT for?</summary>

This isn't for complete beginners who have never used a terminal. You should be comfortable opening a shell, navigating directories, and running basic commands. If you're not there yet, start with the course first.
</details>

<details>
<summary>What level should I be?</summary>

Comfortable with git and the command line. You don't need to be an expert — that's the whole point — but you should know what a commit is and how to cd into a directory.
</details>

<details>
<summary>Is it live or async?</summary>

Live 1:1 sessions with async follow-up. We meet, work on real problems together, and I'm available between sessions for questions and reviews.
</details>

Claude Pro subscription, a terminal, and a GitHub account. For infrastructure tracks, a DigitalOcean account (new accounts get $200 free credit).
