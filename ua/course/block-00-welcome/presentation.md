**Тон**: Енергійний початок. Це — гачок. Зробіть так, щоб їм було цікаво бути тут.

### Слайд 1: Вступний гачок (1 хв)

**Тези:**

> "Підніміть руку, якщо ви користувалися GitHub Copilot. Або ChatGPT для коду. Або вставляли стек-трейс у AI-чатбот і питали 'що не так?'
>
> Чудово. А тепер забудьте все це.
>
> Claude Code — це не автодоповнення. Це не просто вікно чату поруч із редактором. Це повноцінний AI-агент, який може читати ваші файли, виконувати команди, редагувати код і міркувати про весь проєкт. У цьому курсі ми використовуємо підхід terminal-first, бо він дає найчіткішу ментальну модель, але той самий продукт тепер доступний і в IDE, і на десктопі, і в вебі. Це різниця між тим, коли хтось шепоче підказки вам на вухо... і коли досвідчений інженер сідає за ваш стіл і програмує разом із вами."

**Ключовий меседж**: Задайте очікування від самого початку — це кодування на базі агента, а не автодоповнення.

### Слайд 2: Що таке Claude Code насправді? (2 хв)

**Тези:**

- Claude Code починався як terminal-first інструмент від Anthropic — компанії, що стоїть за Claude
- Сьогодні він доступний у терміналі, IDE, десктопі та вебі; цей курс фокусується на терміналі, бо він найкраще розкриває базовий воркфлоу
- Під капотом він має доступ до **інструментів**: може читати файли (Read), шукати (Grep), знаходити файли (Glob), редагувати код (Edit) та запускати Bash-команди
- Уявіть собі AI-пару для програмування з `sudo`-доступом до вашого проєкту (із запобіжниками)
- Він бачить весь репозиторій — не лише відкритий файл
- Він зберігає контекст протягом розмови — ви можете обговорювати архітектуру, покроково дебажити складний баг або планувати цілу фічу

**Аналогія**: "IDE-копайлоти — це як автозаміна на телефоні — корисна, але обмежена тим, що на екрані. Claude Code — це як дзвінок колезі, який може зайти по SSH на вашу машину і реально озирнутися."

**Цікавий факт**: "Ось щось дивовижне — Claude Code частково побудований за допомогою Claude Code. Інженери Anthropic використовували ранні версії інструменту, щоб розробляти пізніші. Це як молоток, який допоміг скувати кращий молоток."

### Слайд 3: Огляд курсу — Карта (2 хв)

**Тези:**

> "Ось подорож, у яку ми вирушаємо. 14 блоків. Кожен будується на попередньому."

Пройдіться по загальній арці:

1. **Блоки 0-2**: Налаштування, дослідження та локальна розробка (сьогодні)
2. **Блоки 3-4**: Планування та внесення реальних змін у код
3. **Блоки 5-6**: Пам'ять, інтелект проєкту та кастомні навички
4. **Блоки 7-8**: Інфраструктура — реальний K8s-кластер на DigitalOcean, з хуками
5. **Блоки 9-10**: MCP-сервери та CI/CD з GitHub Actions
6. **Блоки 11-12**: Субагенти та фінал GitOps з ArgoCD
7. **Блок 13**: Просунуті патерни та що далі

> "До кінця цього курсу ви зробите push коміту, і ArgoCD автоматично задеплоїть ваш застосунок у живий Kubernetes-кластер. Без ручного kubectl. Без SSH на сервери. Чистий GitOps."

### Слайд 4: Курсовий проєкт — ai-coderrank (2 хв)

**Тези:**

- Покажіть скріншот/демо ai-coderrank
- Це дашборд на Next.js 14, що порівнює AI-моделі кодування (GPT-4, Claude, Gemini тощо) за продуктивністю та ціною
- Стек технологій: React 18, TypeScript, Tailwind CSS, Recharts для візуалізації
- Має Docker-збірку з кількома етапами, K8s-маніфести, конфіг ArgoCD, GitHub Actions
- Чому саме цей проєкт? Бо він має все — фронтенд, API-маршрути, контейнеризацію, оркестрацію, CI/CD
- Він реалістичний, але не перевантажений. Такий проєкт можна запустити у стартапі.

> "Ми обрали цей проєкт, тому що він торкається кожного шару сучасного веб-застосунку. Фронтенд-компоненти, API-маршрути, Docker-збірки, Kubernetes-деплойменти, CI-пайплайни. Якщо Claude Code зможе допомогти вам орієнтуватися в цьому та вносити зміни — він допоможе з чим завгодно у вашій повсякденній роботі."

### Слайд 5: Розбивка витрат — Без сюрпризів (1 хв)

**Тези:**

- Підписка Anthropic Pro: $20/місяць — дає доступ до Claude Code
- DigitalOcean droplet (блоки 7+): ~$24/місяць за дроплет із 4 ГБ RAM для k3s
- Разом: приблизно $44/місяць, поки ви активно проходите блоки з інфраструктурою
- Дроплет можна вимкнути одразу після курсу
- Підписка Pro? Ту ви залишите. Повірте.

> "Я хочу бути прозорим щодо витрат. Це не безкоштовний інструмент — але він коштує як підписка на Netflix, а заощадить вам години щотижня. Це не маркетинг — це математика."

### Слайд 6: Що потрібно перед стартом (1 хв)

**Тези:**

Швидкий чеклист — переконайтеся, що в усіх є:

- Термінал, у якому вам комфортно (macOS Terminal, iTerm2, Warp, будь-який Linux-термінал)
- Node.js 18 або новіший (`node --version`) для роботи з проєктом після встановлення
- Git встановлений та налаштований (`git --version`)
- Обліковий запис GitHub (будемо робити форк проєкту)
- Обліковий запис Anthropic із підпискою Pro — зареєструйтеся на [claude.ai](https://claude.ai)
- Використовуйте поточний офіційний шлях встановлення Claude Code: нативний інсталятор на macOS/Linux/WSL, Homebrew на macOS за бажанням або WinGet на Windows

> "Якщо у вас ще немає підписки Pro — зараз чудовий час зареєструватися. Вона знадобиться приблизно через три хвилини, коли ми перейдемо до практичної частини."
