## Стартова лінія

Кожен інструмент має момент, коли він "клікає". Для `git` це був перший раз, коли ви зрозуміли, що можете скасувати будь-що. Для Docker — коли `docker run` дав вам робочу базу даних за дві секунди. Для Claude Code цей момент ось-ось настане — прямо зараз.

Цей блок проведе вас від нуля до першої справжньої розмови з AI-агентом, який може читати ваш код, виконувати команди та пояснювати знайдене. Не автодоповнення. Не чат-бот із текстовим полем. Повноцінний агент, що живе у вашому терміналі.

## Що ми розглянемо

1. **Що таке Claude Code насправді** — і чим він принципово відрізняється від IDE-копайлотів
2. **Встановлення** — одна команда, менше хвилини
3. **Автентифікація** — підключення підписки Anthropic Pro
4. **Перший запуск** — запуск `claude` та орієнтація за допомогою `/help`
5. **Курсовий проєкт** — форк `ai-coderrank` і перший погляд Claude на нього

## Курсовий проєкт: ai-coderrank

Впродовж цього курсу ми працюватимемо з **ai-coderrank** — дашбордом на Next.js 14, що порівнює AI-моделі кодування за продуктивністю та ціною. Це реальний застосунок із реальною інфраструктурою:

- **Frontend**: React 18 + TypeScript + Tailwind CSS + Recharts
- **Backend**: Next.js API routes
- **Контейнеризація**: Docker із багатоетапною збіркою
- **Оркестрація**: Kubernetes-маніфести
- **GitOps**: конфігурація ArgoCD
- **CI/CD**: воркфлоу GitHub Actions

Це не іграшковий проєкт. Це саме та кодова база, яку ви зустрінете на роботі — і ідеальний полігон для вивчення Claude Code.

## Передумови

- Термінал (macOS Terminal, iTerm2 або будь-який Linux-термінал)
- Node.js 18+ встановлений
- Git встановлений та налаштований
- Обліковий запис GitHub
- Підписка Anthropic Pro ($20/місяць)

## Розбивка витрат

Поговоримо про гроші відразу — без сюрпризів:

| Позиція | Вартість |
|---------|----------|
| Підписка Anthropic Pro | $20/місяць |
| DigitalOcean droplet (пізніші блоки) | ~$24/місяць |
| Обліковий запис GitHub | Безкоштовно |
| **Разом** | **~$44/місяць** |

Ви можете вимкнути DigitalOcean droplet одразу після завершення блоків з інфраструктурою. Підписка Pro — єдина постійна витрата, і, чесно кажучи, коли ви побачите, що може Claude Code, скасовувати її не захочеться.
