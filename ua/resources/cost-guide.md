| Складова | Вартість | Обов'язково? |
|----------|----------|--------------|
| Claude Pro | $20/міс | Так |
| DO дроплет (s-2vcpu-4gb) | $24/міс | Блоки 7, 12 |
| Anthropic API ключ | ~$1-5 загалом | Тільки Блок 10 |

DigitalOcean дає новим акаунтам **$200 безкоштовного кредиту** (60 днів) -- достатньо для проходження всього курсу.

## Обов'язкові витрати

### Claude Pro ($20/міс)

- **Включає**: Claude Code у терміналі, IDE, Desktop та Web
- **Моделі**: Sonnet (за замовчуванням) + Haiku
- **Використання**: тарифіковане, спільне з Claude chat -- не безлімітне
- **Opus**: доступний на плані Max ($100/міс), не на Pro
- Деталі на [anthropic.com/pricing](https://anthropic.com/pricing)

### Дроплет DigitalOcean ($24/міс)

- **s-2vcpu-4gb** -- фіксована рекомендація, не обирайте менше
- Нові акаунти отримують **$200 безкоштовного кредиту** (дійсний 60 днів)
- **Видаліть після курсу**, щоб зупинити нарахування -- ваш код живе в git, дроплет є одноразовим

## Який план мені потрібен?

| План | Ціна | Що ви отримуєте в Claude Code |
|------|------|-------------------------------|
| **Pro** | $20/міс | Sonnet + Haiku -- достатньо для всього курсу |
| **Max** | $100/міс | Доступ до Opus, вищі ліміти -- для інтенсивного використання |
| **Team** | різна | Залежить від типу місця -- див. застереження нижче |

<div class="callout-important" markdown="1">

## Як витрачати менше

- `/compact` часто -- стискає розмову, зберігає токени
- `--model haiku` для простих задач
- `/clear` між непов'язаними задачами
- `--max-turns` у скриптах для обмеження ітерацій
- `/cost` у сесії для перевірки витрат

## Варіанти бюджету

### Шлях за $20

Тільки Pro. Пропустіть блоки з DO (7, 12). Зосередьтесь на вивченні можливостей Claude Code, хуків, MCP та агентів локально.

### Шлях за $25-45

Повний курс. Pro + DO дроплет. З безкоштовним кредитом реальна вартість ближче до **~$20** за перші два місяці.

### Командний / корпоративний шлях

Плани Max або Team, компанія покриває інфраструктуру. Використовуйте хмарний акаунт вашої організації для дроплета.
